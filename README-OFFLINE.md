# Luma Studio — Offline Edition v1.0.2

完全离线运行的 Luma Studio，解压即用，无需联网安装依赖。

## 与在线版的区别

| 特性 | 在线版 | 离线版 |
|------|--------|--------|
| 包体积 | ~100 KB（仅源码） | ~80 MB（含 node_modules） |
| 首次启动 | 需联网下载依赖（约 30s） | 直接运行，无需联网 |
| 适用场景 | 有网络环境 | 内网/离线环境 |

## 系统要求

- Windows 7 或更高版本
- [Node.js](https://nodejs.org/) 18 或更高版本（仅需 Node 运行时，不需要 npm 联网）

## 安装方式

### 方式一：一键安装（推荐）

1. 确保已安装 Node.js 18+
2. 双击 **`install-offline.bat`**
3. 程序将自动安装到 `%LOCALAPPDATA%\LumaStudio`，并创建桌面快捷方式
4. 双击桌面「Luma Studio」图标启动

### 方式二：直接运行

双击 **`start-offline.bat`** — 直接从当前目录启动，无需安装

## 离线运行保证

- 本离线版已打包完整的 `node_modules`，启动时**不会**尝试 `npm install`
- 启动脚本仅检查 Node.js 运行时和 `node_modules` 完整性
- 如果 `node_modules` 文件夹缺失，脚本会提示包损坏，不会自动修复（需重新下载完整 ZIP）

## 功能

- **相册浏览**：上传/拖拽照片，缩略图网格，灯箱大图
- **编辑器**：亮度/对比度/饱和度/色相/锐化/模糊，6 种滤镜预设，交互式裁剪，旋转翻转，改分辨率
- **导出**：JPEG/PNG/WebP/AVIF 格式转换与压缩，体积预估，下载到本地
- **选片**：星级评分（1-5星），精选/排除标记，多选批量操作
- **收藏夹**：创建/管理相册，将照片分组
- **幻灯片**：自动播放照片
- **EXIF**：查看/编辑/抹除元数据（支持中文）
- **搜索**：按文件名/格式/星级/标记/相册筛选，多种排序
- **设置**：默认导出格式/质量，缩略图尺寸，主题色

## 快捷键

| 键位 | 功能 |
|------|------|
| `1`-`5` | 给当前照片评分 1-5 星 |
| `0` | 清除评分 |
| `P` | 标记为精选 |
| `R` | 标记为排除 |
| `←` `→` | 灯箱/幻灯片中切换照片 |
| `空格` | 幻灯片中暂停/继续 |
| `Esc` | 关闭灯箱/幻灯片 |

## 卸载

运行 **`uninstall.bat`**，或手动删除安装目录 `%LOCALAPPDATA%\LumaStudio`。

## 技术栈

- **后端**: Express + Node.js
- **图像处理**: sharp (libvips)
- **EXIF**: exifr + piexifjs
- **前端**: 原生 JavaScript + HTML + CSS（零框架依赖）
