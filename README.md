## 安装方式

### 方式一:一键安装 (推荐)

1. 确保已安装 [Node.js](https://nodejs.org/) 18+ (脚本会自动尝试安装)
2. 双击 **`install.bat`** — 自动安装到 `%LOCALAPPDATA%\LumaStudio`,创建桌面快捷方式
3. 双击桌面「Luma Studio」图标启动,浏览器自动打开 http://localhost:7443

### 方式二:直接运行

双击 **`start.bat`** — 自动安装依赖并启动(首次需联网)

### 方式三:手动

```bash
npm install --omit=dev
node server.js
```

## 卸载

双击 **`uninstall.bat`**,或手动删除安装目录。

## 功能

- **相册浏览**: 上传/拖拽照片,缩略图网格,灯箱大图
- **编辑器**: 亮度/对比度/饱和度/色相/锐化/模糊,6 种滤镜预设,交互式裁剪,旋转翻转,改分辨率
- **导出**: JPEG/PNG/WebP/AVIF 格式转换与压缩,体积预估,下载到本地
- **选片**: 星级评分(1-5星),精选/排除标记,多选批量操作
- **收藏夹**: 创建/管理相册,将照片分组
- **幻灯片**: 自动播放照片
- **EXIF**: 查看/编辑/抹除元数据(支持中文)
- **搜索**: 按文件名/格式/星级/标记/相册筛选,多种排序
- **设置**: 默认导出格式/质量,缩略图尺寸,主题色

## 快捷键

| 键位 | 功能 |
|------|------|
| `1`-`5` | 给当前照片评分 1-5 星 |
| `0` | 清除评分 |
| `P` | 标记为精选 |
| `R` | 标记为排除 |
| `←` `→` | 灯箱/幻灯片中切换照片 |
| `空格` | 幻灯片中暂停/继续 |
| `Esc` | 关闭灯箱/幻灯片 |
